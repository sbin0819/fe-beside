## 기본 세팅
next install

```bash
yarn add next react react-dom
```

`pakage.json` 에 `scrpits` 추가

```json
"scripts": {
  "dev": "next dev",
  "build": "next build",
  "start": "next start",
  "lint": "next lint"
}
```

타입스크립트 관련 라이브러리 설치

```
yarn add -D @types/react @types/react-dom @types/node typescript
```
